Gakuto Deshimaru（学人デシマル）是UtaUtaUtau制作的UTAU虚拟歌手。本仓库是学人デシマル χ 日文VCV音源的镜像站。音频和oto均没有修改。

根据Gakuto Deshimaru的[用户协议](license.md)，在这个音源停止配布前，只能完整地二次配布本音源，禁止修改。二次配布时，必须正确地注明音源名称和作者。
